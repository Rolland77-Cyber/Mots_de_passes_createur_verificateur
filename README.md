1. Analyse de mots de passe (Force Checker)

Fonctionnalités :

Vérification de la longueur
Détection majuscules / minuscules
Présence de chiffres
Présence de symboles
Vérification dans une liste de mots de passe courants


2. Gestion d’accès simple

Fonctionnalités :

Création de compte (username + mot de passe)
Stockage des mots de passe en SHA-256
Authentification simple (login)
Base utilisateurs sauvegardée en JSON



Technologies utilisées :

Domaine	      Choix
Langage      	Python 3
Sécurité	    SHA-256 (hashlib)
Stockage	    JSON simple

Pyton 3 nécessaire pour l'installation.
